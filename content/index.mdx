## Web アプリ学習コースの進め方
基本的には、「a1. GitHub」から、順番に単元を進めます。この学習カリキュラムを進める前に、必ず以下のルールを確認しておいてください。

### 受講者資格
Java カリキュラムを修了、10 クラス以上を用いた Java のコンソールアプリを作成する技術や知識が身についていること（事前に基本情報技術者試験を合格しているとより学習しやすいです）。
